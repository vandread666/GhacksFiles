Résumé

Microsoft a publié un correctif cumulatif (Update Rollup 1) pour Microsoft Windows XP. Un correctif cumulatif est un ensemble cumulatif de correctifs, de correctifs de sécurité, de mises à jour critiques et de mises à jour qui sont réunis afin de faciliter leur déploiement. Update Rollup 1 pour Windows XP est un package qui contient de nombreuses mises à jour critiques précédemment publiées pour Windows XP. Pour plus d'informations sur les mises à jour critiques incluses dans ce correctif cumulatif, cliquez sur les numéros ci-dessous pour afficher les articles correspondants dans la Base de connaissances Microsoft.

Informations sur l'installation

La version standard de ce correctif cumulatif prend en charge les commutateurs d'installation suivants :
  • /? : affiche la liste des commutateurs d'installation.
  • /u : utilise le mode sans assistance.
  • /f : force la fermeture des autres programmes lors de l'arrêt de l'ordinateur.
  • /n : ne sauvegarde pas les fichiers à supprimer.
  • /o : remplace les fichiers OEM sans avertissement.
  • /z : ne redémarre pas une fois l'installation terminée.
  • /q : utilise le mode silencieux (sans intervention de l'utilisateur).
  • /l : répertorie les correctifs installés.
  • /x : extrait les fichiers sans exécuter le programme d'installation.
Pour vérifier que le correctif cumulatif a été installé, utilisez l'outil Qfecheck.exe ou Microsoft Baseline Security Analyzer (MBSA). Pour plus d'informations sur l'outil Qfecheck.exe, cliquez sur le numéro ci-dessous pour afficher l'article correspondant dans la Base de connaissances Microsoft.
282784

Remarque Lorsque vous installez ce correctif cumulatif, toutes les entrées existantes dans l'outil Ajouter ou supprimer des programmes sont supprimées pour les mises à jour critiques comprises dans ce correctif cumulatif. Par conséquent, certains outils peuvent signaler que certaines mises à jour incluses ne sont pas installées. Par exemple, les solutions de gestion du changement de Marimba Software utilisent les entrées de Registre de l'outil Ajouter ou supprimer des programmes qui sont associées à chaque mise à jour critique pour confirmer l'installation de la mise à jour. Par conséquent, les solutions de gestion du changement de Marimba Software signaleront de manière incorrecte que les mises à jour critiques incluses dans ce correctif cumulatif sont manquantes.

Informations sur le déploiement

Pour installer le correctif cumulatif sans intervention de la part de l'utilisateur, utilisez la commande suivante :
windowsxp-kb826939-x86-enu.exe -u -q
Pour installer le correctif cumulatif sans forcer le redémarrage de l'ordinateur, utilisez la commande suivante :
windowsxp-kb826939-x86-enu.exe -z
Remarque Vous pouvez combiner ces commutateurs en une seule commande.

Pour plus d'informations sur le déploiement de ce correctif cumulatif à l'aide des services de mises à jour logicielles, reportez-vous au site Web de Microsoft à l'adresse suivante (en anglais) :
http://www.microsoft.com/windowsserversystem/updateservices/evaluation/previous/default.mspx
(http://www.microsoft.com/windowsserversystem/updateservices/techinfo/previous/susdeployment.mspx)

Nécessité d'un redémarrage

Vous devez redémarrer votre ordinateur après avoir appliqué ce correctif cumulatif.

Informations sur la suppression

Pour supprimer ce correctif cumulatif, utilisez l'outil Ajouter ou supprimer des programmes du Panneau de configuration pour supprimer 826939. Remarque Lorsque vous installez ce correctif cumulatif, toutes les entrées existantes dans l'outil Ajouter ou supprimer des programmes sont supprimées pour les mises à jour critiques comprises dans ce correctif cumulatif. Lorsque vous supprimez ce correctif cumulatif, les entrées précédentes de l'outil Ajouter ou supprimer des programmes pour les mises à jour critiques individuelles comprises dans ce correctif cumulatif sont restaurées.

Les administrateurs système peuvent utiliser l'utilitaire Spuninst.exe pour supprimer ce correctif cumulatif. L'utilitaire Spuninst.exe se trouve dans le dossier %Windir%\$NTUninstallKB826939$\Spuninst. Il prend en charge les commutateurs d'installation suivants :
  • /? : affiche la liste des commutateurs d'installation.
  • /u : utilise le mode sans assistance.
  • /f : force la fermeture des autres programmes lors de l'arrêt de l'ordinateur.
  • /z : ne redémarre pas une fois l'installation terminée.
  • /q : utilise le mode silencieux (sans intervention de l'utilisateur).

Informations sur le remplacement des correctifs de sécurité

Ce correctif cumulatif ne remplace pas d'autres correctifs de sécurité.
