Restore shortcuts to the Accessories group

AccRestore - restores missing shortcuts to the Accessories group

This utility restores the missing shortcuts to the Start Menu Accessories group. The Accessories group includes sub-folders namely Accessibility, Communications, System Tools, Entertainment.

Download AccRestore v2.0 ~14Kb (for Windows XP only)
