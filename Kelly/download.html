Download PSP Thumbnail Handler

PSP Thumbnail Handler 2.1, a freeware utility, is available for download.

  • 32-bit edition for Windows XP, Windows Server 2003, and newer versions (214 KB EXE file)
  • 32-bit edition for Windows 2000 (1.09 MB EXE file)
  • 32-bit edition for Windows 98 and Windows Me (1.09 MB EXE file)
  • x64 edition (316 KB EXE file)

System Requirements - 32-bit Edition

  • Pentium/166-based or compatible system, or better
  • One of the following Microsoft operating systems:
    • Windows XP
    • Windows Server 2003
    • Windows 2000 with Internet Explorer 5.5 or newer
    • Windows Me
    • Windows 98 with Internet Explorer 5.5 or newer
  • 2 MB of available hard drive space

Note: The ability to view image dimensions in Windows Explorer's Details view requires Windows XP, Windows Server 2003, or future versions.

Note: PSP Thumbnail Handler uses GDI+ 1.0, a standard component of Windows XP, Windows Server 2003, and future Windows versions. Because this component is not part of previous versions of Windows, a separate installation package that includes GDI+ 1.0 is available above for those operating systems.

System Requirements - x64 Edition

  • AMD 64-based or compatible system
  • One of the following Microsoft operating systems:
    • Windows XP Professional x64 Edition
    • Windows Server 2003 x64 Edition
  • 2 MB of available hard drive space
